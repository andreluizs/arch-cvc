## Arch DEV - Scripts

> install.sh  
> pos-install.sh  
> dev-install.sh

## Install Base System
> ```shell 
> # Warning this action will be erase your disk
> bash <(curl -s -L bit.do/arch-dev/install.sh) 
> ```
## Pos-Install
> After install base system, reboot and inside your `$HOME` execute:
> ```shell 
> # Desktop Environment
> bash pos-install.sh
>
> # Tools for Developer
> bash dev-install.sh
>```
